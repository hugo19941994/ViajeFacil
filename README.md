Proyecto integrador de 1º de la [Universidad Europea de Madrid (UEM)](http://madrid.universidadeuropea.es/) con [Unisys](http://www.unisys.es/) - 2015

## Compilar

```bash
$ cd source
$ qmake
$ make
$ make clean
```
